# EventCrawler

Bizouk event monitor for Raspberry.

## Install
```bash
curl -fsSL https://raw.githubusercontent.com/mopsoner/EventCrawler/main/install.sh -o install.sh && chmod +x install.sh && ./install.sh
```

## Run
```bash
cd ~/EventCrawler && chmod +x run.sh && ./run.sh
```
