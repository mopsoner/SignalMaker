Admin SignalMaker

La page admin est intégrée au frontend léger et disponible sur admin.html.

Le frontend écoute par défaut sur le port 8090 et appelle l’API sur le port 5000.

Champs Kraken pris en compte

L’admin executor lit et sauvegarde les clés de configuration Kraken ci-dessous dans le fichier .env et dans les réglages persistants SQLite.

ChampUtilisation
EXECUTION_EXCHANGEChoisir kraken ou kraken_pro pour utiliser Kraken au lieu de Binance.
KRAKEN_BASE_URLURL API Kraken, par défaut https://api.kraken.com.
KRAKEN_API_KEYClé API Kraken, affichée comme champ mot de passe dans l’admin.
KRAKEN_SECRET_KEYSecret API Kraken, affiché comme champ mot de passe dans l’admin.
